# Medicine-Management-System
#####Project Description:
仁爱药品管理系统是一个基于B/S架构、Boostrap+JQuery+AdminTLE框架+AJAX和Java EE技术而实现跨终端、跨平台的在线药品管理软件，实现药店营业员对日常销售和药品信息的管理。
